Ansible Collection Kubernetes Spices

The playbooks could be run using the Ansible Execution environment.

The collection builds ansible execution environment and publishes them at quay.io/kameshsampath/kubernetes-spices-ansible-runner.

This page is not complete, a working example will be added soon

Running a playbook

To run a playbook you need to create directory structure as explained in Runner Input Directory Hierarchy. Once you have that, you can run the playbook like,

run.sh
docker run -it --rm --name=ansible-runner \
  -v `pwd`:/project \
  quay.io/kameshsampath/kubernetes-spices-ansible-runner
