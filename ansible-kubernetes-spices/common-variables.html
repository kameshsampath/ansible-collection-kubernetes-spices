Ansible Collection Kubernetes Spices

The following ansible variables are common to all the roles depending on th contexts where they are applied.

Common Variables

Table 1. Common Ansible Variables
Name Description Default Value

k8s_contexts

The list of Kubernetes contexts from the $KUBECONFIG file. These contexts will be looped over when applying other tasks such as Istio, Knative etc., To add the values to the list use the command:

kubectl config get-contexts | awk 'NR>1{print $2}'

This parameter accepts a list e.g.

- kind-vanilla
- bar
- foo
