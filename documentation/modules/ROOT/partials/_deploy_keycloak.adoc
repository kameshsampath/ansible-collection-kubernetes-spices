Keycloak tasks allows you to configure the https://www.keycloak.org[Keycloak] installation.

[#keycloak-ansible-vars]
== Ansible Variables

.Tekton Ansible Variables
|===
| Name | Description | Default Value

| deploy_keycloak
| To deploy Keycloak or not, accepts `true` or `false`
| false

| keycloak_namespace
| The namespace to install Keycloak
| keycloak

| keycloak_admin_user
| The Keycloak admin user
| admin

| keycloak_admin_password
| The Keycloak admin user password
| admin

|===
