Operator Lifecycle Manager(OLM) tasks allows you to configure the https://github.com/operator-framework/operator-lifecycle-manager[OLM]. Please check https://opertorhub.io[Operator Hub] for various community operators that can be installed using OLM.

[#olm-ansible-vars]
== Ansible Variables

.OLM Ansible Variables
|===
| Name | Description | Default Value

| deploy_olm
| To deploy OLM or not, accepts `true` or `false`
| false

| olm_version
| The OLM version to install
| v0.18.2

| olm_namespace
| The namespace to install OLM
| olm

|===
