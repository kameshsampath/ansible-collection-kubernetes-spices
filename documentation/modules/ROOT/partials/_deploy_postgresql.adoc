PostgreSQL tasks allows you to configure the http://www.postgresql.org/[PostgreSQL] database.

[#postgresql-ansible-vars]
== Ansible Variables

.Postgresql Ansible Variables
|===
| Name | Description | Default Value

| deploy_postgresql
| To deploy Postgresql or not, accepts `true` or `false`
| false

| postgresql_namespace
| The namespace to install Postgresql
| db

| postgresql_admin_user
| The Postgresql admin user
| postgres

| postgresql_admin_password
| The Postgresql admin user password
| postgres

| postgresql_db
| The Postgresql database
| demodb

|===
